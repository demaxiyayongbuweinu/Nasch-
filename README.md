红绿灯三种模式：1.关闭（就是没有）  2.常规红绿灯（一红一绿）3.智能红绿灯（检测密度，给水平主干道多个绿灯，比如三绿一红）
智能交通灯的意思其实就是监测密度，然后决定给水平还是竖直开红绿灯，比如说给水平开了一次绿灯之后。我发现水平道路的这个密度还是比竖直道路的密度要大。这个时候我就继续给水平道路开一次绿灯，但是要设置一个上限，比如说给某一条道路连续开三次绿灯之后，就必须给这条道路红灯
